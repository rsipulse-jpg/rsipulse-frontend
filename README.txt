Files included:
- lib/main.dart
- lib/api.dart
- lib/models.dart

Steps:
1. Copy into your Flutter project (replace main.dart if needed).
2. Add to pubspec.yaml:
   dependencies:
     http: ^1.2.2
     shared_preferences: ^2.2.3
3. Run app, set backend base URL if different.
